# STAR D0-Tagged Jet EEC Analysis

Energy-Energy Correlators are expected to exhibit the Dead Cone Effect, a suppression of gluon radation in a cone around an asymptotically free heavy quark. This effect has been observed at the LHC by the ALICE and LHCb collaborations, but due to kinematic constraints a clearer measurement is expected at RHIC energies. Through the analysis of STAR data, we hope to demonstrate this by performing the measurement.  

This repository consists of three main parts:
- Simulation code used to study the feasibility and requirements to perform the measurement.
- Data analysis (StMakers) code run on RCF to scan over STAR data.
- Analysis scripts to process the final data gathered/trimmed from the larger STAR dataset.
